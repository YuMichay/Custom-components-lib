# Custom-components-lib


## 🌐 English

Custom-components-lib is a custom component library based on React and MUI, packaged as an npm module with Storybook support and built with Webpack.

### 📄 Task

The task for this project is described in detail in the [Google Document](https://drive.google.com/file/d/1C148FRnWfXVoRDslDWcYac3bEhebdIAV/view).

### 🚀 How to Run the App

Install the library via npm:

```sh
npm install custom-components-lib-yumi
```

Or via yarn:

```sh
yarn add custom-components-lib-yumi
```

### 🛠️ Usage

Example usage of the Button component:

```sh
import React from "react";
import { Button } from "custom-components-lib-yumi";

function App() {
  return <Button variant="contained">Click me</Button>;
}

export default App;
```

### 📖 Documentation

To view all available components, run Storybook:

```sh
npm run storybook
```
